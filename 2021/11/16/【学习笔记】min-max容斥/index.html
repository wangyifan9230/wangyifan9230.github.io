min−maxmin-maxmin−max 容斥

对于一个集合 SSS , 设 max(S)max(S)max(S) 为集合中最大的元素, min(S)min(S)min(S) 为集合中最小的元素, 则有:

max(S)=∑T⊆S(−1)∣T∣−1min(T)max(S) = \sum_{T \subseteq S}{(-1)^{|T|-1}min(T)} max(S)=T⊆S∑​(−1)∣T∣−1min(T)

反过来也成立.

这个式子的原理在于:

max({x})=min({x})=xmax(\{x\}) = min(\{x\}) = x max({x})=min({x})=x

证明:

构造容斥系数 f(x)f(x)f(x) 使得原式成立.

max(S)=∑T⊆Sf(∣T∣)min(T)max(S) = \sum_{T \subseteq S}{f(|T|)min(T)} max(S)=T⊆S∑​f(∣T∣)min(T)

考虑集合 SSS 中第 kkk 小的数会被统计到的贡献, 就是枚举哪些集合的最小值为这个数.

∑i=0n−k(n−ki)f(i+1)=[n−k=0]\sum_{i=0}^{n-k}{\binom{n-k}{i}f(i+1)} = [n-k = 0] i=0∑n−k​(in−k​)f(i+1)=[n−k=0]

把 n−kn-kn−k 换一下, 更形式化的表示.

∑i=0x(xi)f(i+1)=[x=0]\sum_{i=0}^x{\binom{x}{i}f(i+1)} = [x = 0] i=0∑x​(ix​)f(i+1)=[x=0]

记 F(x)=f(x+1),G(x)=[x=0]F(x) = f(x+1), G(x) = [x = 0]F(x)=f(x+1),G(x)=[x=0] .

则有,

∑i=0x(xi)F(i)=G(x)\sum_{i=0}^x{\binom{x}{i}F(i) = G(x)} i=0∑x​(ix​)F(i)=G(x)

二项式反演,

F(x)=∑i=0x(−1)x−i(xi)G(i)=(−1)xF(x) = \sum_{i=0}^x{(-1)^{x-i} \binom{x}{i} G(i)} \\ =(-1)^x F(x)=i=0∑x​(−1)x−i(ix​)G(i)=(−1)x
